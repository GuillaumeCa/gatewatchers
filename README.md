# Gate Watchers

## Controls

- Z/S: Move forward/backward
- Q/D: Move left/right
- A/E: Roll left/right
- Ctrl/Space: Strafe down/up
- Shift: Enable Boost
- X: Switch between ship and FPS
- T: Toggle Travel mode
- M: Toggle System Map
- Esc: Show menu
